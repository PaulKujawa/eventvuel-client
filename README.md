# GraphQL client for Tickermaster

## Run locally
Start with `npm start`.

|name               |purpose|
|---                |---|
|Now                |Deployment|

|Typescript & Babel |JS static typing|

|Vuetify            |UI framework|
|SASS               |CSS preprocessor|
|PostCSS            |CSS auto-prefixer|
