# GraphQL client for Ticketmaster

## Contribute
* Start with `npm start`
* GraphQL-TypeScript-Schema can be updated with
```
apollo client:codegen \
  --endpoint http://localhost:4000 \
  --includes src/**/*.gql \
  --target typescript
```
